Sobre mim
Afinal de contas, quem sou eu?

Naquele periodo, era bem escasso o acesso a informação. Atualmente estou cursando Analise e Desenvolvimento de Sistema e já criei alguns sites para amigos e conhecidos. Tenho conhecimento em diversas áreas, desde manutenção de computador, redes, desenvolvimento de sites e sistemas (tanto back-end como front-end), serviços de hospedagem em rede e etc...

Desenvolvedor WEB

Desenvolvedor Web é o profissional responsavel pelo desenvolvimento de sites e sistemas da internet. Hoje um desenvolvedor web pode elaborar projetos desde ecommerce, até sistemas de emissão de NF. Não importa o tamanho de seu negócio, se quiser ficar conhecido na rede, você precisará de um sistema web para divulgar a sua marca. O mundo hoje gira em torno das tecnologias desenvolvidas e o poder de divulgação esta em um clique Se voce não tem um site, não se precupe... Fale comigo agora mesmo e divulgaremos a sua marca o quanto antes.

Hardware

Hardware é o nome tecnico dado a todo equipamento fisico que faça parte de um computador... Nessa familia, podemos classificar os hardwares e periféricos. Hardware é todo componente interno do computador... Seja memória, HD, fonte, cooler, processador, placa de video e etc. Periféricos é todo componente plugavel externamente do computador... Ou seja, monitor, mouse, teclado, impressora, scanner e etc... Todo periferico faz parte dos Hardware, mas nem todo hardware é periferico.

Software

Software é o nome dado a todo programa executado pelo seu computador. O próprio sistema operacional é um programa, independente de ser windows/linux/macOS. Os programas são necessários para poder se utilizar o computador, seja para navegar na internet ou formalizar um documento de texto. Os programas existem para facilitar a nossa vida, uma vez que o computador só é capaz de reconhecer comandos em zero e um, os programas fazem a tradução do que queremos fazer para o que o computador é capaz de entender...

Redes

Rede de computador é um conjunto de dois ou mais dispositivos conectado através de uma serie de protocolos e conjuntos de regras que permitem a comunicação entre sí. Os dispositivos que controlam essa conexão são chamados de "nós de rede". O exemplo mais comum de rede é a internet. Com ela você é capaz de acessar e enviar dados para basicamente qualquer computador do mundo. Ela também é a maior rede existente atualmente. Quando nos referimos a internet, é indiferente se o acesso é pelo computador, celular, tablet e etc. No final, o protocolo de rede é o mesmo
